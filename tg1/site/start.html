Quick Start

Last updated: Mar 19th, 2017

Overview

This content present a brief practical introduction on the conceptual framework for the Biodiversity Informatics community. To clarify how each stakeholder can take advantage of the conceptual framework, we organize the content according to four main stakeholders:

DQ Profiler
  • Who: experts on DQ and/or as specific domain that uses biodiversity-related data.
  • Interest: formalizing the way DQ is handling in a specifi domain.
Developer
  • Who: experts on to develop technical solutions for DQ.
  • Interest: formalizing techniques and tools used for DQ and generate standardized ouputs.
Data User
  • Who: experts on a specific domain which uses biodiversity-related data.
  • Interest: assessing the quality of data and their fitness for use.
Data Holder
  • Who: institution or person that holdes, manage and curate biodiversity-related data.
  • Interest: improving the quality and the fitness for use of data with efficience.
